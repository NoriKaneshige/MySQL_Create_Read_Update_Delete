# MySQL_Creating_Database_and_Tables

## Start the CLI:
```
mysql-ctl cli; 
```
## List available databases:
```
show databases; 
```
## The general command for creating a database:
```
CREATE DATABASE database_name; 
```
## To drop a database:
```
DROP DATABASE database_name;
```
